# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [3.1.4] - 2025-12-29

### Added
- Initial npm package release as `beacon-icons`
- 90+ SVG icon components as React components
- Support for token-based sizes (`xs`, `sm`, `rg`, `md`, `lg`, `xl`, `2xl`)
- Support for numeric pixel sizes
- TypeScript support with full type definitions
- Icons use `currentColor` for easy theming

### Features
- React 18 and 19 compatible
- TypeScript support
- Flexible sizing (tokens or pixels)
- CSS variable support for icon sizes
- Full TypeScript type definitions
